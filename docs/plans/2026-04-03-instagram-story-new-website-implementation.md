# Instagram Story New Website Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Produce a premium vertical Instagram Story announcing the launch of the new AM Estetica Dental website.

**Architecture:** Build one short vertical motion piece using the site's existing visual language: dark background, gold accents, premium typography, and restrained motion. The output should work as a muted-first story and end with the website URL clearly readable.

**Tech Stack:** Existing website visual assets, motion/video tooling available in repo or local environment, export to vertical video format.
